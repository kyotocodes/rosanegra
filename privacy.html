Política de Privacidade - RosaNegra Bot

Última atualização: Maio de 2025

1. Coleta de Dados

O RosaNegra Bot processa dados temporários como IDs de usuários, nomes de personagens e interações com eventos e vendas. Nenhum dado pessoal sensível é coletado.

2. Finalidade

As informações são utilizadas exclusivamente para gerenciar atividades de uma organização de GTA RP, incluindo registros, vendas e eventos.

3. Armazenamento e Segurança

Os dados são processados de forma temporária e segura. Não armazenamos informações sensíveis em bancos de dados externos.

4. Compartilhamento de Dados

Não compartilhamos informações com terceiros. Os dados são utilizados apenas internamente para execução dos comandos do bot.

5. Direitos dos Usuários

Usuários podem solicitar remoção de seus dados a qualquer momento, entrando em contato com a equipe responsável pela organização.

6. Contato

Para dúvidas ou solicitações, entre em contato com os administradores do bot via Discord.
